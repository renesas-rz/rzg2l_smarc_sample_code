libgpiod output example

This is the gpio input application with libgpiod using RZ/G2L and PMOD Module

Build

To use libgpiod in RZ-G, in yocto, add the recipe to image, this can be done in local.conf with 

    IMAGE_INSTALL_append = “ libgpiod libgpiod-tools”
You must install a Yocto SDK that contains the libgpiod library for RZG2L. 
    $ source /opt/poky/3.1.5/environment-setup-aarch64-poky-linux
    $ make

run

    $ ./libgpiod-input
