Capture Image using openCV

This is the image capture from MIPI camera using openCV and RZ/G2L

Build

To use openCV in RZ-G, in yocto, add the recipe to image, this can be done in local.conf with 

    IMAGE_INSTALL_append = " opencv"‚Äù
You must install a Yocto SDK that contains the openCV library for RZG2L. 
    $ source /opt/poky/3.1.5/environment-setup-aarch64-poky-linux
    $ cmake -DCMAKE_TOOLCHAIN_FILE=../rzg2_aarch64-gnu.toolchain.cmake .
    $ make

run

    $ ./openCV-captureImage /dev/video0
